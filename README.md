動作環境 node.js 学内ICT室

実行方法
1. node.jsのインストール
2. nodeによりserver.jsを実行(ターミナルで実行)
3. localhost:8000に接続（chrome, firefox, safari等）
4. ログイン画面にてID、パスワードを入力（ユーザーがない場合は新規作成へ）、決定してホーム画面に遷移
5. ホーム画面からメニュー画面、予定追加画面に遷移することができる
